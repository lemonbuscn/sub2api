LemonFly 文档

一个 API Key，接入 Claude 和 GPT 等主流 AI 模型。LemonFly 为你提供稳定的 API 中转、订阅套餐、额度管理与按量付费能力。

开始使用

第一次接入：5 分钟接入指南  |  已有 Key：客户端接入  |  返回控制台

LemonFly 是什么

LemonFly 是一个 AI API 订阅服务平台，让你用一个 API Key 就能调用 Claude、GPT 等多种模型。平台负责账号管理、智能调度、负载均衡和计费，你只需关注开发。

  • 多模型统一接入 — 一个 Key 调用 Claude Sonnet/Opus、GPT 系列等主流模型
  • 智能调度 — 自动选择最优账号，支持粘性会话和故障转移
  • 灵活计费 — 订阅套餐 + 按量付费两种模式，按 Key 控额度

核心能力

🔑 一个 Key 通用

同一个 API Key 可调用多种模型，无需为每家上游分别注册账号

📊 精准计费

Token 级别用量追踪，日限额 / 周限额 / 月限额精细控制

⚡ 智能调度

自动选择最优账号，支持粘性会话，保证连续对话体验

🛡️ 并发控制

按用户和账号限制并发数，防止资源滥用

适合谁用

  • 想用 Claude Code / Codex CLI 但不想分别开 Anthropic、OpenAI 账户
  • 团队共用 AI 巜具，希望统一计费、按 Key 控额度
  • 需要稳定的 API 中转服务，避免网络不稳定问题

Claude Code 接入

Claude Code 是 Anthropic 官方的 AI 编程工具。通过 LemonFly 接入，只需修改两个配置。

配置方式

在终端中设置环境变量：

export ANTHROPIC_BASE_URL="https://你的域名"
export ANTHROPIC_API_KEY="sk-你的key"

或者在 ~/.bashrc / ~/.zshrc 中持久化。

验证

claude "Hello, 你好"

如果正常返回响应，说明接入成功。

OpenAI SDK 接入

LemonFly 兼容 OpenAI API 格式，可直接使用 OpenAI 官方 SDK。

Python

from openai import OpenAI

client = OpenAI(
    base_url="https://你的域名/v1",
    api_key="sk-你的key"
)

response = client.chat.completions.create(
    model="claude-sonnet-4-20250514",
    messages=[{"role": "user", "content": "Hello!"}]
)
print(response.choices[0].message.content)

Node.js

import OpenAI from 'openai';

const client = new OpenAI({
    baseURL: 'https://你的域名/v1',
    apiKey: 'sk-你的key'
});

const response = await client.chat.completions.create({
    model: 'claude-sonnet-4-20250514',
    messages: [{ role: 'user', content: 'Hello!' }]
});
console.log(response.choices[0].message.content);

错误排查

401 Unauthorized

API Key 无效或已过期。检查：

  • Key 是否正确复制（以 sk- 开头）
  • Key 是否已过期或被禁用
  • Header 格式是否正确：Authorization: Bearer sk-xxx

403 Forbidden

权限不足。检查：

  • Key 绑定的分组是否包含目标模型
  • 账号是否被禁用

429 Too Many Requests

请求频率超限。检查：

  • 是否超出并发限制
  • 是否超出日限额 / 周限额 / 月限额
  • 稍等片刻后重试

502 / 503 服务不可用

上游服务暂时不可用。平台会自动重试其他账号，如果持续出现请联系管理员。

常见问题

套餐和按量付费有什么区别？

套餐适合稳定高频使用，包含日/周/月限额；按量付费适合低频、临时或需要弹性控制成本的场景。

适合 Claude Code 使用吗？

完全适合。只需设置 ANTHROPIC_BASE_URL 和 ANTHROPIC_API_KEY 环境变量即可。

额度会自动重置吗？

是的。日限额每天、周限额每周一、月限额每月 1 日自动重置。

支持流式输出吗？

支持。在请求中设置 "stream": true 即可使用 SSE 流式输出。

企业或团队能使用吗？

可以。MAX 套餐更适合高强度开发和团队协作场景。管理员可以为不同用户分配不同的 API Key 和额度。

数据安全吗？

LemonFly 作为中转网关，不存储你的对话内容。所有请求实时转发到上游模型服务。
